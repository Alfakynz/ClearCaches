## 🚀 Objective

The development of the CLI has just begun. This project allows you to delete system, user, and application caches with customizable parameters, such as the size of caches to delete.


#### 🔨 Xcode
```bash
cd macos
xcodebuild -scheme clrc -sdk macosx -derivedDataPath ./build
./build/Build/Products/Release/clrc
```
